CONFIGURATION

The configuration parameters from the table below have default values in the directory <MONGOOSE_DIR>/conf/defaults.json

Note
  • User can specify Mongoose configuration parameters as JVM system properties via JAVA_TOOL_OPTIONS environment variable. For example:
    export JAVA_TOOL_OPTIONS="-Dstorage.addrs=node74.cloud.storage.com,node75.cloud.storage.com,node76.cloud.storage.com,node77.cloud.storage.com -Ditem.data.size=1B -Dload.threads=100 -Dload.limit.time=5m"
  • The meaning of "container" is different for different storage types / cloud storage APIs:
    • S3: the container means the bucket
    • Atmos: has an effect if storage.http.fsAccess=true
    • Swift: the container means the container
    • Filesystem load: the container means the target directory
      • may contain several directories, for example: "a/bb/ccc"
      • may contain leading slash: "/tmp/123"
